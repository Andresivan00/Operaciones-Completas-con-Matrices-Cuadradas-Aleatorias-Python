# Operaciones Completas con Matrices Cuadradas Aleatorias (Python)

Programa educativo que genera dos matrices cuadradas **A** y **B** de tamaño aleatorio (3×3 a 6×6), y realiza **todas las operaciones clásicas** de álgebra de matrices:

### Qué hace exactamente este código

1. **Define un tamaño n aleatorio** entre 3 y 6
2. **Llena A y B** con números aleatorios del 1 al 10
3. Calcula automáticamente:
   - **C = A + B** → suma elemento por elemento
   - **D = A − B** → resta elemento por elemento
   - **E = A × B** → multiplicación real de matrices (¡no elemento por elemento!)
4. **Ordena la matriz E** de **mayor a menor** usando el algoritmo **burbuja** (aunque sea ineficiente, es muy didáctico)
